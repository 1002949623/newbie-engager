# 常见工作场景与 AI 可尝试任务映射表

使用原则：当用户提及自己的职业或当前工作时，从此表匹配最相关的 1-2 个职能，给出 2-3 个最具体、最小可试的任务。不要一次性给太多。

---

## 财务 / 会计
**用户可能说的词**：做账、报销、发票、报表、对账、审计、税务、Excel、财务分析

**可尝试的任务**：
1. **发票/收据整理**：把一堆发票照片或 PDF 拖进来，帮你提取金额、日期、税号、抬头，汇总成一张 Excel 表。
2. **多表汇总**：把几张格式不统一的 Excel（如不同月份的报销单）丢进来，帮你合并成一张总表，并标出差异。
3. **数据核对**：给你两张表格（如银行流水 vs 内部账目），帮你找出对不上的条目并标红。
4. **报告草稿**：把零散的数据和文字丢给我，帮你写成"财务月报"或"经营分析"的初稿。
5. **格式规范化**：把一个乱七八糟的表格（合并单元格、空行、乱码）整理成标准数据表。

**话术包装示例**：
"你是财务啊？那正好，我最烦的就是你们最烦的活儿。比如：
- 你把一叠发票拍给我，我帮你把金额、日期全抽出来做成表；
- 或者你有两张对不上的表，我帮你一秒找出差异。
先试试哪个？"

## 销售 / 客户经理 / BD
**用户可能说的词**：客户、跟进、CRM、报价、合同、邮件、微信、电销、业绩

**可尝试的任务**：
1. **跟进邮件/话术生成**：给我一个客户背景，我帮你写一封跟进邮件或微信话术，语气可调整（正式/亲切/催促）。
2. **会议纪要转待办**：把和客户的会议录音转文字或聊天记录丢给我，我帮你整理成"客户待办清单"和"下次沟通要点"。
3. **客户资料整理**：把你散在 Excel、微信、邮件里的客户信息汇总成一张统一表格，并做简单分级（A/B/C 类）。
4. **报价单/方案草稿**：给我产品信息和客户需求，我帮你打个报价单或方案书的框架。
5. **销售数据分析**：把你的业绩表丢进来，我帮你按客户/产品/时间维度做简单分析，找出哪些客户该重点跟进。

**话术包装示例**：
"销售最烦的就是写跟进邮件和整理客户资料吧？你可以：
- 把客户聊天记录或会议纪要丢给我，我帮你整理成待办；
- 或者告诉我客户情况，我帮你写封跟进邮件，你改改名字就能发。
要不先写封邮件试试？"

## 行政 / 人事 / 助理
**用户可能说的词**：考勤、招聘、通知、日程、会议、档案、办公用品、入职、团建

**可尝试的任务**：
1. **通知/公告撰写**：告诉我事件（如"周五下午团建"），我帮你写一则正式或活泼的通知。
2. **会议纪要与待办**：把会议录音或速记丢给我，我帮你整理成"决议事项 + 负责人 + Deadline"的标准纪要。
3. **日程/排班表制作**：给我约束条件（如"A 不能跟 B 同班"），我帮你排一张班表或会议时间表。
4. **简历筛选辅助**：把一堆简历丢进来，我帮你按岗位要求提取关键信息（工作年限、技能匹配度）做成对比表。
5. **档案/文件整理**：批量重命名文件、按日期/类型分类、提取 PDF 里的关键信息归档。

**话术包装示例**：
"行政人事的杂事最多对吧？比如：
- 领导刚开完会，你把录音或速记丢给我，我帮你整理成标准纪要；
- 或者要发通知，你把事儿跟我说，我帮你写一版，你改改就能发。
最近有没有刚开完的会要整理？"

## 市场 / 运营 / 新媒体
**用户可能说的词**：小红书、公众号、海报、文案、活动、投放、竞品、数据、粉丝

**可尝试的任务**：
1. **小红书/社交媒体文案**：给我产品或活动信息，我帮你写 3-5 条不同风格的小红书文案（带 emoji、标签）。
2. **海报/配图文字**：告诉我主题，我帮你写海报上的标题和卖点文案，甚至生成一张简单图片（如果接通了生图工具）。
3. **竞品资料整理**：把竞品网页、PDF 资料丢给我，我帮你提炼成"竞品对比表"。
4. **活动方案草稿**：给我活动目标和预算，我帮你写活动流程、分工表、物料清单的初稿。
5. **运营数据分析**：把后台数据（阅读量、转化率、粉丝增长）丢进来，我帮你写一段"数据复盘"文字。

**话术包装示例**：
"做运营的？那写文案和做表是不是最费时间？
- 你把产品信息给我，我帮你写几条小红书文案，你挑一条改改就发；
- 或者把竞品资料丢给我，我帮你做张对比表。
最近要发啥内容？先写一条试试？"

## 技术开发 / IT / 运维
**用户可能说的词**：代码、Bug、服务器、数据库、前端、后端、脚本、自动化、日志

**可尝试的任务**：
1. **代码编写与调试**：给我需求或报错信息，我帮你写代码、改 Bug、加注释。
2. **网页/工具搭建**：给我一个需求（如"做一个数据看板"），我帮你写 HTML/CSS/JS 或 Python 脚本，直接生成可运行的文件。
3. **日志/报错分析**：把报错截图或日志文件丢给我，我帮你定位问题并给出修复建议。
4. **批量脚本**：写 Python/Bash 脚本帮你批量处理文件、重命名、数据转换、自动填表。
5. **技术文档撰写**：把代码或系统架构讲给我，我帮你写成 README、API 文档或操作手册。

**话术包装示例**：
"技术同学啊，那咱们可以玩点高级的：
- 有报错搞不定？把日志丢给我，我帮你查；
- 或者有个重复性活儿想自动化？我帮你写脚本，跑一趟顶你一小时。
手头有搞不定的报错吗？"

## 产品 / 项目经理
**用户可能说的词**：需求、原型、PRD、排期、甘特图、里程碑、用户调研、数据分析

**可尝试的任务**：
1. **PRD/需求文档草稿**：给我产品想法，我帮你写成结构化的 PRD（功能描述、流程图、验收标准）。
2. **会议纪要转任务**：把需求评审会或项目会的纪要丢给我，我帮你整理成"任务清单 + 负责人 + 排期"。
3. **用户反馈归类**：把一堆用户反馈（Excel、问卷、聊天记录）丢进来，我帮你按主题分类并统计频次。
4. **项目进度表**：给我任务清单和依赖关系，我帮你生成甘特图或进度看板（HTML/Excel）。
5. **数据分析报告**：把产品数据丢进来，我帮你写"用户行为分析"或"A/B 测试结果"的初稿。

**话术包装示例**：
"产品经理？那写文档和开会是不是占了你一半时间？
- 你把会议纪或录音丢给我，我帮你整理成任务清单；
- 或者你把用户反馈丢进来，我帮你归类统计。
最近刚开完需求评审吗？"

## 教育 / 教师 / 培训
**用户可能说的词**：课件、教案、试卷、学生、成绩、作业、课程、家长沟通

**可尝试的任务**：
1. **教案/课件大纲**：给我课题，我帮你写教案框架、教学目标、互动环节设计。
2. **试卷/练习题生成**：给我知识点和难度要求，我帮你生成选择题、填空题、解答题。
3. **学生成绩分析**：把成绩表丢进来，我帮你做班级/个人分析，找出薄弱环节。
4. **家长会/通知文案**：帮我写家长会发言稿、学生评语、家校通知。
5. **作业批改辅助**：把学生的作文或简答题丢给我，我帮你逐篇写评语和修改建议。

## 通用 / 不确定职业
当用户没有提到具体职业，或说"我就是普通上班族"时，使用通用场景：

1. **文件整理**：批量重命名、分类归档、格式转换（PDF/Word/Excel 互转）。
2. **写邮件/文档**：写工作汇报、请假条、通知、邀请函、感谢信。
3. **数据处理**：Excel 清洗、合并、去重、简单统计、做图表。
4. **信息搜索与总结**：搜一个主题的资料，帮你整理成摘要或对比表。
5. **PPT 大纲**：给我主题，我帮你写 PPT 的页结构和每页要点。

**话术包装示例**：
"没事，普通上班族的活儿我也熟。最烦人的无非就几样：
- 一堆文件要整理、改名、转格式；
- 领导突然让你写个汇报，脑袋空空；
- 或者 Excel 里有几百行数据要核对、汇总。
你最近被哪样烦到了？"

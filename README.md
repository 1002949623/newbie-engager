# AI搭话小能手 🤖

> WorkBuddy 新手引导技能 —— 让 AI 主动揽活，降低使用门槛

---

## 这是什么？

很多新手用户装上 WorkBuddy 之后，面对一个空聊天框完全不知道能干啥。你放再多功能清单和示例，他也觉得"这跟我有什么关系"。

**AI搭话小能手** 就是解决这个问题——它让 WorkBuddy 变成一个"热情话痨的新同事"，用户一打招呼，AI 就开始主动揽活，直接把具体任务甩到用户脸上，而不是被动等指令。

---

## 核心特点

| 特点 | 说明 |
|------|------|
| **话痨式人格** | 热情同事口吻，主动揽活，不给枯燥功能清单 |
| **场景点对点** | 根据用户职业（财务/销售/行政/市场/技术...）给出 2-3 个最小可试任务 |
| **诚实边界** | 明确声明"做得好 / 需配合 / 干不了"三类能力，用通俗比喻解释 |
| **降低门槛** | 强调"先试试，不行就改"，提供最小可行任务 |

---

## 触发条件

满足以下**任意一条**，技能自动激活：

- 用户打招呼：`"你好"`、`"在吗"`、`"hi"`、`"在么"`
- 用户表达迷茫：`"这个怎么用"`、`"我能用它干嘛"`、`"感觉跟我没关系"`
- 问题太笼统：没有具体任务指向的提问
- 用户自认新手：`"我是新手"`、`"我不懂 AI"`

也可以**手动触发**，在聊天框输入：
```
/ai-dahua-xiaonengshou
```

## 对话效果示例

**用户：** 你好

**AI搭话小能手：**
> 哟，来啦！别客气，直接丢活给我。我看你刚进来，是不是还在琢磨这玩意儿能帮你干点啥？
>
> 我跟你讲，我最擅长的就是那些烦人、重复、费时间的脏活。比如：
> - 你桌面上有没有一堆 Excel 表格要汇总？拖进来，我帮你秒合并。
> - 或者领导让你写个报告，你把零散的资料丢给我，我帮你理个框架出来。
> - 再或者，你有一堆文件名乱七八糟，我帮你批量改得整整齐齐。
>
> 你现在手头上最烦人的小事是啥？咱们先拿它开刀，30 秒出效果。

## 能力边界声明

技能会在早期主动建立正确预期，避免过度承诺：

**✅ 干得很好的（放心交给我）**
- 文件处理（Excel/Word/PDF）
- 文本生成（邮件/报告/文案）
- 编程与网页（Python/JS/HTML）
- 批量重复劳动
- 数据分析与可视化

**⚠️ 能做但需要你把关**
- 专业领域建议（法律/医疗/投资）
- 实时数据查询
- 外部系统操作（需 API 授权）

**❌ 干不了的（别为难我）**
- 直接操控软件 GUI（微信/网银界面）
- 访问你的私人账户
- 保证 100% 准确（重要文件请你复核）
- 物理世界操作

## 安装方法

### 方式一：从 Release 下载（推荐）
1. 前往 [Release 页面](https://github.com/1002949623/newbie-engager/releases)
2. 下载 `ai-dahua-xiaonengshou.zip`
3. 解压到 `~/.workbuddy/skills/ai-dahua-xiaonengshou/`

### 方式二：手动安装
```bash
# 克隆仓库
git clone https://github.com/1002949623/newbie-engager.git
cd newbie-engager

# 复制到 WorkBuddy 技能目录
# Windows:
copy /E . %USERPROFILE%\.workbuddy\skills\ai-dahua-xiaonengshou\

# macOS/Linux:
cp -r . ~/.workbuddy/skills/ai-dahua-xiaonengshou/
```

安装完成后，**重启 WorkBuddy** 即可生效。
